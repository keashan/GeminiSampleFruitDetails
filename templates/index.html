Gemini AI App Development - MVP

There are few methods to develop apps using Gemini AI. I will be able to use four methods and python language to build usable apps.

  • Freeform prompt apps.

This kind of app can send a simple prompt with or without image to the Gemini ai and get an answer. As an example, we can send an image to Gemini and ask for a description about it.

You can check the "Fruit" page of this website to get an idea about this kind of app. There, you can upload an image of a fruit to get the name and a small description about it. If you uploaded an image that has more than one fruit or no fruit on it, you would get an error message.

  • Structured prompt apps.

You can train the ai to follow a pattern using this kind of app. As an example, we can give few product names and descriptions about them to train the style we used to write product descriptions. Then when we give a new product name, it will generate a description that follows the pattern we used.

You can check the "Poem" page of this website to get an idea about this kind of app. It will generate a poem when you give a topic.

  • Chat prompt apps.

Using this kind of app, we can have a conversation with Gemini. During the setup, we can assign a role and characters to the app and then it follows them when you are chatting with it.

You can check the "Chat" page of this website to get an idea about this kind of app. I assigned a "Train Lover" role to it. So, AI will reply to you about any questions about trains. It will remind you he is an expert at trains and does not know other stuff if you ask anything else.

  • Chat agent.

Using this method, we can train a model with your data and the power of Gemini.

You can check the floating button with chat icon on any page to experience this. There I created a chatbot and a chat agent to provide details about this website and about the services I provide. You can chat with that bot and get an answer.
